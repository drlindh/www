L﻿åset till lägenhetens ytterdörr är medlemmens ansvar. Det finns dock möjlighet att använda samma låssystem som föreningen använder till t.ex. porten. Om man vill byta sitt lägenhetslås till iLOQ 5S, så kan man kontakta föreningens låssmed. Enklast via formuläret nedan.

E﻿fter ifyllt formulär kontaktar Alexanderssons Lås dig för att göra upp en tid när det passar att genomföra låsbytet. Notera att det endast är låssmeden som kan utfärda nycklar till lägenhetslås, fastighetsskötaren kan inte tillverka sådana nycklar, och har inte heller tillgång till någon huvudnyckel.
